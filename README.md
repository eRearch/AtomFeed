## 这是什么

AtomFeed 是一个面向**科研工作者**的每日文献情报服务。

每天早上 8 点，你的邮箱里会出现一封邮件——里面只有**和你研究方向真正相关**的最新论文，附带 AI 中英双语总结和关键实验数据提取。

不是算法推给你的，是你自己定义的。
